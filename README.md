# react-web-resize
An react version of scroller inspired by iscroller.

### Prerequirement ###

* `install [nodeJs && npm]`(https://nodejs.org/)

### Install package ###

* `[sudo] npm install -g gulp`
* `npm install`

### Build ###

* `gulp test`

### Run ###

* npm run
then open your browser and type `localhost:9527`
